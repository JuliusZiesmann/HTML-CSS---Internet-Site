Diese Lösung zur Konvertierung von Zahlen von einem Zahlensystem in ein anderes habe ich mit C# geschrieben.
Release Version: RechnerJZ - 01.12 Portable für Windows x-86
Dieses auf C# basierende Projekt demonstriert, wie der Eignungstest für IT der Agentur für Arbeit funktioniert

Mit Hilfe dieses Programms können Sie nachvollziehen, wie das Eignungstest Agentur für Arbeit verläuft und welche Fragen gestellt werden können.
Der Test besteht aus 50 Fragen, die verschiedene Bereiche abdecken. Darunter Allgemeinwissen, Mathematik, Logik und Informatik.

Der Zeitansatz für diese Prüfung beträgt 45 Minuten.
